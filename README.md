# TF-Model-Service (Only a demo)
Flask microservice of TensorFlow models

It has been tested with all 10 images. Everything works.  经过测试，一切运行正常。
### Run
```bash
python TF-Model-Service.py
```
### Test
```bash
python model.py
```

  body: raw
